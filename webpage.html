Draft · CPLC Throttleable Hotfire Challenge

2. Lumped Parameter Model for TOAD Vehicle
Draft · Physics Engine Architecture

Outlines a physics engine decomposing complex fluid systems into nodes and links. It utilizes a hybrid solver transitioning between Ideal Gas and Bulk Modulus formulations, paired with a Variable-Step Implicit Euler integrator to handle dynamic stiffness.

3. S-TADPOLE Throttle Architecture Draft
Draft · Controller Implementation

Proposes a decoupled Lead/Follow valve architecture to improve throttle system robustness. The oxidizer circuit actively tracks chamber pressure targets while the fuel circuit independently maintains a constant mixture ratio.

Comprehensive documentation of the GNC and simulation workflow for the ASTRAv2 VTOL prototype. Features the derivation of an 18-state Multiplicative Extended Kalman Filter (M-EKF), digital filter sequences, and a 3-loop cascaded LQRi tracking controller.
